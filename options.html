Crossword to Anki

Pick where new cards go. The extension talks to Anki via the AnkiConnect add-on, so Anki needs to be running when you save a card.

Field mapping

Tell the extension which field on the note type holds the clue, the answer, and any optional extras.

Setup checklist

  1. Install the AnkiConnect add-on in Anki.
  2. In Anki: Tools → Add-ons → AnkiConnect → Config. Add this extension's ID to webCorsOriginList: 
    chrome-extension://YOUR_EXTENSION_ID
  3. Restart Anki, then click Refresh above.
  4. Pick your deck and note type, map the fields, save.
  5. While solving on nytimes.com, press Ctrl/Cmd+Shift+A to capture the active clue.
